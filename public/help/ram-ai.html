Ram AI Helper

An optional assistant inside the Signpack Maker web editor. It helps you work faster on Traffic Control signpacks — you stay in control of what gets saved.

Open editor (login required)

Optional, not required

Upload, save, and export signpacks work the same with Ram AI off. The helper only runs when you click a button.

Your API key stays private

The browser never talks to Ram AI directly and never sees RAM_AI_API_KEY. Only the Signpack Maker server calls /api/ram-ai/models, /helper, /status, and /cancel. Choose a model from the dropdown (display names only).

Shared queue

Ram AI handles one generation at a time. Extra requests wait in a queue — the editor shows busy / queued status and lets you cancel.

Advisory suggestions

Ram AI does not auto-edit your pack. Copy answers into tooltip text, notes, or JSON yourself after reviewing them.

What it can help with

  • Explain signpack fields and Traffic Control rules
  • Review signs.json for possible issues
  • Suggest tooltips, notes, and textline ideas
  • Convert colors to integer format
  • Draft a pack README / description
  • Custom questions with optional editor context you choose
